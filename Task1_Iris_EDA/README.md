# Task 1: Exploring and Visualizing the Iris Dataset

## Objective
Learn how to load, inspect, and visualize a dataset to understand
data trends, distributions, and relationships between features.

## Dataset
- **Name:** Iris Dataset
- **Source:** Built-in via Seaborn
- **Size:** 150 samples, 5 columns
- **Classes:** Setosa, Versicolor, Virginica

## Models Applied
No model — EDA and visualization only

## Key Results & Findings
- Petal length and petal width are the strongest species predictors
- Dataset is perfectly clean with zero missing values
- Setosa is clearly separable from other two species
- Petal length and petal width are 96% correlated
